    strong_edge_coloring()

    Partition undirected qubit edges into 8 non-adjacent sets (strong edge coloring).

    For a square lattice, the strong chromatic index is 8. We construct 8 buckets (0..3 for horizontal edges, 4..7 for vertical edges) such that within each bucket no two edges share a vertex and no two edges are at graph-distance 2 in the line graph (i.e., they do not form a length-2 path).

    Construction

    Let each node have integer coordinates (x, y). - For a horizontal edge, choose the left endpoint (x, y) as canonical and set bucket = (x + 2 * (y & 1)) % 4 (buckets 0..3) - For a vertical edge, choose the lower endpoint (x, y) as canonical and set bucket = 4 + ((y + 2 * (x & 1)) % 4) (buckets 4..7)

    Returns:

    Type Description
    list[list[tuple[int, int]]]

    A list of 8 lists of undirected edges (u, v) with u < v. Each list is a strong matching.
