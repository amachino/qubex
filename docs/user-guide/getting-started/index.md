# Getting started

This section helps you install Qubex, prepare configuration, and run your first measurement.

## Recommended path

1. [Installation](installation.md)
2. [Quickstart](quickstart.md)
3. [Concepts](../concepts/overview.md)

## Before you begin

- Qubex expects chip configuration and parameter files. If you are working with hardware, obtain these from your lab setup and pass them via `config_dir` and `params_dir`.
- If you are prototyping without hardware, you can use `mock_mode=True` to skip device control while still validating schedules and data flow.
